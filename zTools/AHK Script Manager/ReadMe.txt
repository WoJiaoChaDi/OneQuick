脚本管理器本身也是一个ahk程序，启动后会运行/scripts 目录下的所有ahk脚本，但是如果脚本文件名前面带有“!”感叹号，则不会自动运行，可以通过菜单运行。
1.“关闭脚本”则自动结束对应的脚本进程，
2.“编辑脚本”可以调用默认编辑器对脚本内容进行修改，方便调试，
3.“重载脚本”用于修改脚本后重新载入新的脚本。
4.“添加脚本”会自动打开/scripts 目录，方便复制新的ahk脚本进去。
5.“我添加了新脚本”则会重新运行/script 目录下面的所有脚本，感叹号开头的除外。
6.“退出”是指推出脚本管理器，并不影响正在运行的ahk脚本。
