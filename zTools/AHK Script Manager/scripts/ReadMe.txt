------WPS中---
^+f::	使用WPS表格时，按Ctrl+Shift+F 返回首页并打开搜索

---chrome中---
!1::	使用chrome，按alt+1  查看chrome上一处搜索
!2::	使用chrome，按alt+2  查看chrome下一处搜索
^f::	使用chrome，按ctrl+f  自动搜索选中文字
!v::	使用chrome时，按Alt+v,查看框架源代码
^+x::	使用chrome时，按Ctrl+Shift+X,查看框架源代码

---idea中---
#w::	在eclipse里面，win+w打开主干合并代码的文件目录
^+CapsLock::	Ctrl+Shift+CapsLock --》 Ctrl + PgUp（上一个视图）
^CapsLock::	Ctrl+CapsLock --》 Ctrl + PgDn（下一个视图）
+!D::	使用eclipse时，按Alt+Shift+D执行Ctrl+Shift+O  导包
^B::	使用eclipse时，按Ctrl+B执行Ctrl+O
!`::	使用eclipse时，按Alt+`执行Alt+/
^Tab::	使用eclipse时，按ctrl+tab在两个括号间切换
Insert::	在eclipse的svn对比工具时，按insert将不同点替换成旧内容
]::	使用eclipse时，按]  查看svn同步的下一处不同
[::	使用eclipse时，按[  查看svn同步的上一处不同

---文件资源管理器中---
#t::	新建文件夹，并以当前日期自动命名

---任何时候---
!m::	alt+m 切换当前窗口的最大化/最小化	
!n::	alt+n 最小化当前激活窗口
#Esc::!F4	Win+Esc映射为 alt+F4
!a::	alt+A 选中光标所在行所有内容，并把光标定位到末尾
+!1::	shift+alt+1 --> ctrl+home
+!2::	shift+alt+1 --> ctrl+end
 #!e::	win+alt+e  打开所选中的文件路径所在的文件夹，并定位到该文件
~esc::	按esc切换输入法
#WheelUp::	 win+上滚轮 透明窗口
#WheelDown::	win+下滚轮  透明窗口
~PrintScreen::	按4下PrintScreen息屏
CapsLock & LButton::	按住CapsLock和左键拖动当前窗口
#v:: 		新增热字符串,是在本脚本的末尾新增字段（会自动替换热键重名的热字符串）
